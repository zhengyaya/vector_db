# 向量数据库学习路径

## 阶段1：基础 - 语义搜索系统 📚

**目标**：理解向量 embedding 和相似度搜索的核心概念

### 项目：个人文档语义搜索工具

**需求**：
1. 导入你日常记录的笔记/文章（10-20篇）
2. 用自然语言搜索相关文档
3. 支持按类型筛选（技术/生活/学习）
4. 显示相似度分数和原文片段

**核心技能**：
- 文本 → 向量
- 向量检索
- 距离计算 (cosine/L2)

---

## 阶段2：进阶 - RAG 知识问答系统 🤖

**目标**：结合 LLM 实现问答系统

### 项目：本地文档知识问答助手

**需求**：
1. 读取本地文档（PDF/TXT/MD）
2. 构建知识库，向量存储
3. RAG 方式提问，返回原文引用
4. 支持多文档交叉引用

**核心技能**：
- 文档切分（chunking）
- 向量入库
- 上下文召回
- prompt 工程

---

## 阶段3：综合 - AI 客服系统 🎯

**目标**：综合应用，构建完整产品

### 项目：企业知识库客服系统

**需求**：
1. 企业知识库管理（FAQ、产品文档）
2. 对话历史记忆
3. 多轮对话支持
4. 多集合管理、元数据过滤
5. 混合检索、重排序

**核心技能**：
- 多集合管理
- 元数据过滤
- 混合检索
- 重排序（rerank）

---
